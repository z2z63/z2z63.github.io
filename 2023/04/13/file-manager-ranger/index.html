ж–Үд»¶йў„и§Ҳ

йў„и§Ҳе°ұжҳҜеңЁдёҚжү“ејҖж–Үд»¶зҡ„жғ…еҶөдёӢпјҲдёҚдҪҝз”Ёй»ҳи®ӨиҪҜд»¶жү“ејҖе®ғпјүеҝ«йҖҹжҹҘзңӢж–Үд»¶еҶ…е®№
еңЁwindowsдёҠе®һзҺ°ж–Үд»¶йў„и§ҲпјҢеҸҜд»Ҙжү“ејҖExploreзҡ„йў„и§ҲеҠҹиғҪпјҢдҪҶжҳҜиҝҷз§ҚеҠҹиғҪе®һйҷ…дҪҝз”ЁдёӯжҜ”иҫғйёЎиӮӢпјҢеӣ дёәе®ғеҸӘиғҪйў„и§ҲжһҒе°‘зұ»еһӢзҡ„ж–Үд»¶пјҲдҫӢеҰӮtxtпјүпјҢиҖҢдё”иҝҳдјҡеҚ з”ЁзӘ—еҸЈеҫҲеӨ§зҡ„з©әй—ҙ е®һйҷ…дёҠпјҢж—ҘеёёдҪҝз”ЁдёӯйҒҮеҲ°зҡ„ж–Үд»¶зұ»еһӢжҳҜйқһеёёеӨҚжқӮзҡ„пјҢжҜ”еҰӮиҜҙзәҜж–Үжң¬зҡ„ж–Үд»¶

  • txt
  • еҗ„з§ҚиҜӯиЁҖзҡ„жәҗж–Үд»¶пјҢдҫӢеҰӮc,cpp,h,py,java,js,html,css,csвҖҰ
  • ж•°жҚ®дәӨжҚўж–Үд»¶пјҢдҫӢеҰӮjson, xml, scvвҖҰ
  • й…ҚзҪ®ж–Үд»¶,дҫӢеҰӮconf, ini, cfg, yamlвҖҰ
  • ж—Ҙеҝ—ж–Үд»¶пјҢдҫӢеҰӮlog,outзӯү
  • жІЎжңүжү©еұ•еҗҚзҡ„зәҜж–Үжң¬
  • еҗ„з§Қеҗ„ж ·зј–з Ғж–№ејҸзҡ„зәҜж–Үжң¬

еҸҜи§ҒзәҜж–Үжң¬ж–Үд»¶зҡ„зұ»еһӢе°ұжңүйқһеёёеӨҡз§ҚпјҢиҖҢдёҚеҗҢзәҜж–Үжң¬ж–Үд»¶зҡ„з”ЁйҖ”дёҚеҗҢпјҢжңҹжңӣзҡ„жҳҫзӨәж–№ејҸд№ҹдёҚеҗҢпјҢжҜ”еҰӮиҜҙ

  • еӨ§йғЁеҲҶж–Үд»¶йғҪеҸӘйңҖиҰҒејҖеӨҙеҚҒеҮ иЎҢпјҢдёҚйңҖиҰҒиҜ»еҸ–е®Ңж•ҙзҡ„ж–Үд»¶йҖ жҲҗиҝҮеӨҡзҡ„зЈҒзӣҳи®ҝй—®жөӘиҙ№жҖ§иғҪ
  • еҗ„з§ҚиҜӯиЁҖзҡ„жәҗж–Үд»¶гҖҒй…ҚзҪ®ж–Үд»¶гҖҒж•°жҚ®дәӨжҚўж–Үд»¶жңҖеҘҪиғҪжңүиҜӯжі•й«ҳдә®пјҢе№¶дҪҝз”Ёзӯүе®Ҫеӯ—дҪ“жҳҫзӨә
  • ж—Ҙеҝ—ж–Үд»¶пјҢжңҖеҘҪй«ҳдә®жҳҫзӨәWarningзӯүдҝЎжҒҜ
  • еҚідҪҝжІЎжңүжү©еұ•еҗҚпјҢд№ҹжңҖеҘҪиғҪеӨҹиҜҶеҲ«еҮәе®ғжҳҜзәҜж–Үжң¬е№¶йў„и§Ҳ

еҸҰеӨ–пјҢиҝҳжңүеҫҲеӨҡеҗ„з§Қеҗ„ж ·зҡ„ж–Үд»¶зұ»еһӢйңҖиҰҒдҪҝз”ЁдёҚеҗҢзҡ„йў„и§Ҳж–№ејҸпјҢдҫӢеҰӮ

  1. officeзҡ„doc/docx, ppt/pptx, xls/xlsxж–Үд»¶еҝ«йҖҹйў„и§Ҳ
  2. еӣҫзүҮjpg, png, svgзӯүзӯүзҡ„йў„и§Ҳ
  3. htmlпјҢmdж–Үд»¶зҡ„йў„и§Ҳ

жңҖзү№ж®Ҡзҡ„пјҢдәҢиҝӣеҲ¶ж–Үд»¶пјҢзұ»еһӢйқһеёёеӨҡпјҢеёёи§Ғзҡ„жңү

  1. еҗ„з§Қе№іеҸ°дёҠзҡ„еҸҜжү§иЎҢ
  2. еҗ„з§Қе№іеҸ°дёҠзҡ„еҠЁжҖҒеә“пјҢйқҷжҖҒеә“
  3. ж•°жҚ®ж–Үд»¶пјҲж•°жҚ®ж јејҸжңүзәҰе®ҡпјү
  4. еҺӢзј©ж–Үд»¶пјҢеӯҳжЎЈж–Үд»¶ вҖҰ

еҸҜи§Ғз§Қзұ»е®һеңЁжҳҜеӨӘеӨҡдәҶпјҢеҫҲеӨҡж–Үд»¶з®ЎзҗҶеҷЁеҫҖеҫҖеҸӘиғҪеҒҡеҲ°йў„и§ҲдёҖйғЁеҲҶзҡ„ж–Үд»¶
rangerйҖҡиҝҮз®ҖеҚ•зҡ„й…ҚзҪ®е°ұиғҪеҒҡеҲ°иҜҶеҲ«ж–Үд»¶е№¶йў„и§ҲеӨ§йғЁеҲҶзҡ„ж–Үд»¶пјҢеҸҜд»ҘйҖҡиҝҮдҝ®ж”№%rangerdir/data/scope.shжқҘеҗҜз”ЁзәҜж–Үжң¬ж–Үд»¶д»ҘеӨ–зҡ„йў„и§ҲпјҲйў„и§ҲдәҢиҝӣеҲ¶пјү scope.shдјҡиҮӘеҠЁеҜ»жүҫе·Із»Ҹе®үиЈ…зҡ„е…¶д»–иҪҜд»¶е№¶дҪҝз”Ёе®ғпјҢжҜ”еҰӮиҜҙ

  • file з”ЁдәҺжЈҖжөӢж–Үд»¶зұ»еһӢ
  • chardet дёҖдёӘpythonзҡ„жЁЎеқ—пјҢз”ЁдәҺжЈҖжөӢж–Үд»¶зј–з Ғж–№ејҸ
  • python-bidi з”ЁдәҺжҳҫзӨәд»ҺеҸіеҲ°е·Ұд№ҰеҶҷзҡ„ж–Үеӯ—пјҢдҫӢеҰӮйҳҝжӢүдјҜиҜӯ
  • img2txt з”ЁдәҺжҳҫзӨәеӣҫзүҮзҡ„ASCIIеӯ—з¬Ұз”»
  • w3mimgdisplay / ueberzug / mpv / iTerm2 / kitty / TerminologyзӯүзӯүпјҢз”ЁдәҺеӣҫзүҮйў„и§Ҳ
  • convert жҳҜimagemagickе·Ҙе…·зҡ„дёҖйғЁеҲҶпјҢз”ЁжқҘиҮӘеҠЁж—ӢиҪ¬еӣҫзүҮе№¶йў„и§ҲsvgеӣҫзүҮ
  • ffmpegthumbnailer з”ЁдәҺжҳҫзӨәи§Ҷйў‘зҡ„е°ҒйқўпјҲзј©з•Ҙеӣҫпјү
  • highlight / bat / pygmentize з”ЁдәҺд»Јз Ғзҡ„иҜӯжі•й«ҳдә®
  • atoll / bsdtar / unrar / unzip / zipinfo / sed / tar / 7z з”ЁжқҘйў„и§ҲеӯҳжЎЈе’ҢеҺӢзј©ж–Үд»¶
  • lynx / w3m / elinks з”ЁжқҘйў„и§Ҳhtmlж–Үд»¶пјҲжёІжҹ“htmlпјү
  • pdftotext / mutoolйў„и§Ҳpdfзҡ„ж–Үжң¬пјҢpdftoppmжҠҠpdfиҪ¬жҲҗеӣҫзүҮйў„и§Ҳ
  • djvutxt йў„и§Ҳж–Үжң¬зұ»еһӢзҡ„DjVuж–Үд»¶, ddjvuжҠҠDjVuж–Үд»¶иҪ¬жҲҗеӣҫзүҮйў„и§Ҳ
  • calibre / epub-thumbnailer йў„и§Ҳepubж–Үд»¶пјҲз”өеӯҗд№Ұж–Үд»¶ж јејҸпјүзҡ„еӣҫзүҮ
  • transmission-show йў„и§ҲBitTorrentж–Үд»¶пјҲз§Қеӯҗж–Үд»¶пјүзҡ„дҝЎжҒҜ
  • mediainfo / exiftool йў„и§ҲеӘ’дҪ“ж–Үд»¶
  • odt2txt йў„и§ҲOpen DocumentпјҲејҖж”ҫж–ҮжЎЈж јејҸпјҢжҳҜж–ҮжЎЈж јејҸзҡ„ж ҮеҮҶпјҢofficeзҡ„word, ppt, excelд№ҹж”ҜжҢҒиҝҷдәӣж јејҸжүҖд»ҘиғҪйў„и§Ҳпјү
  • python / jq йў„и§Ҳjsonж–Үд»¶
  • fontimage йў„и§Ҳеӯ—дҪ“ж–Үд»¶

иҝҷз§ҚжңәеҲ¶зҡ„дјҳзӮ№е°ұжҳҜпјҢеҸӘиҰҒжңүеҸҜд»Ҙз”Ёзҡ„е·Ҙе…·пјҢrangerе°ұдјҡдҪҝз”Ёе®ғжқҘйў„и§ҲпјҢиҖҢдёҚйңҖиҰҒд»»дҪ•йўқеӨ–й…ҚзҪ®пјҢеҰӮжһңжІЎжңүпјҢrangerд№ҹиғҪжӯЈеёёе·ҘдҪңпјҢеҸӘжҳҜе°‘дёҖдёӘеҠҹиғҪ

жү“ејҖж–Үд»¶

еҪ“е…үж ҮеңЁдёҖдёӘж–Үд»¶ж—¶пјҢеҸҜд»Ҙиҫ“е…ҘlдҪҝз”Ёй»ҳи®Өзҡ„ж–№ејҸжү“ејҖдёҖдёӘж–Үд»¶пјҢиҝҷдёӘж“ҚдҪңеҫҲз¬ҰеҗҲvimзҡ„ж“ҚдҪңзҡ„зӣҙи§ү д№ҹеҸҜд»Ҙиҫ“е…ҘrпјҢе®ғдјҡеј№еҮәдёҖдёӘжү“ејҖж–№ејҸзҡ„еҲ—иЎЁпјҢеҸҜд»Ҙиҫ“е…Ҙж•°еӯ—йҖүжӢ©жү“ејҖзҡ„ж–№ејҸпјҢд№ҹеҸҜд»ҘзӣҙжҺҘеӣһиҪҰд»ЈиЎЁй»ҳи®Өж–№ејҸ жӯӨеӨ–пјҢеҪ“е…үж ҮеӨ„дәҺзӣ®еҪ•ж—¶пјҢиҫ“е…Ҙlдјҡиҝӣе…ҘиҝҷдёӘзӣ®еҪ•пјҢиҖҢиҫ“е…ҘrеҸҜд»ҘйҖүжӢ©дёҖдёӘж–№ејҸжү“ејҖиҝҷдёӘзӣ®еҪ•жӯӨж—¶еҸҜд»ҘйҖүжӢ©з”Ёdolphinжү“ејҖиҝҷдёӘзӣ®еҪ•

й«ҳзә§еҠҹиғҪ

иҝҮж»Ө

linuxзҡ„shellеңЁж–Үд»¶зі»з»ҹдёӯеҜјиҲӘж—¶пјҢдҪҝз”ЁtabиҝӣиЎҢиЎҘе…ЁжҳҜеҫҲж–№дҫҝзҡ„пјҢжҜ”еҰӮиҜҙдёҖдёӘж–Үд»¶зү№еҲ«й•ҝпјҢжҲ‘дёҚжғіе®Ңж•ҙиҫ“е…ҘпјҢеҸӘйңҖиҰҒиҫ“е…ҘеүҚйқўеҮ дёӘеӯ—з¬ҰпјҢеҶҚжҢүtabпјҢе°ұиғҪиҮӘеҠЁиЎҘе…ЁдәҶпјҢиҝҷд№ҹжҳҜshellеҘҪз”Ёзҡ„дёҖдёӘйҮҚиҰҒеҺҹеӣ 
зұ»дјјең°пјҢrangerд№ҹжңүж №жҚ®ж–Үд»¶еҗҚзҡ„еүҚеҮ дёӘеӯ—з¬Ұеҝ«йҖҹзЎ®е®ҡж–Үд»¶зҡ„еҠҹиғҪ иҫ“е…ҘfпјҢејҖе§ӢжҹҘжүҫж–Үд»¶пјҢеҸӘйңҖиҰҒиҫ“е…Ҙж–Үд»¶зҡ„дёҖйғЁеҲҶеӯ—з¬ҰпјҢrangerдјҡиҮӘеҠЁжҠҠе…үж Үз§»еҠЁеҲ°жҗңзҙўеҲ°зҡ„ж–Үд»¶дёҠпјҢеҰӮжһңиғҪеӨҹе”ҜдёҖзЎ®е®ҡж–Үд»¶пјҢrangerдјҡзӣҙжҺҘжү“ејҖе®ғпјҲжү“ејҖж–Үд»¶жҲ–зӣ®еҪ•пјүпјҢзҶҹз»ғдҪҝз”ЁиҝҷдёӘеҠҹиғҪпјҢеҸҜд»Ҙеҝ«йҖҹеңЁж–Үд»¶ж ‘дёӯеҜјиҲӘпјҒ
еҰӮжһңfжҹҘжүҫзҡ„з»“жһңдёҚе”ҜдёҖпјҢеҸҜд»Ҙз”ЁnеҲ°иҫҫдёӢдёҖдёӘжҗңзҙўз»“жһңпјҢз”ЁNеҲ°иҫҫдёҠдёҖдёӘжҗңзҙўз»“жһң
еҰӮжһңfжҗңзҙўзҡ„з»“жһңжңүеҫҲеӨҡпјҢи·іиҪ¬иө·жқҘд№ҹжҳҜзӣёеҪ“йә»зғҰзҡ„пјҢиҖҢдё”жңүдәӣжғ…еҶөдёӢпјҢйңҖиҰҒжҹҘзңӢдёҖзұ»ж–Үд»¶пјҢдҫӢеҰӮ

  1. жүҫеҲ°еҗҢдёҖжү©еұ•еҗҚзҡ„ж–Үд»¶
  2. жүҫеҲ°е‘ҪеҗҚе…·жңүеҗҢдёҖи§„еҫӢзҡ„ж–Үд»¶ иҝҷз§Қж—¶еҖҷеҸҜд»Ҙиҫ“е…ҘzzпјҢејҖеҗҜжЁЎзіҠжҹҘжүҫжЁЎејҸпјҢе®ғи·ҹfзҡ„еҢәеҲ«е°ұжҳҜпјҢдјҡиҝҮж»ӨжҺүдёҚеҢ№й…Қзҡ„з»“жһңпјҢиҖҢдё”дёҚдјҡиҮӘеҠЁиҝӣе…Ҙзӣ®еҪ•
    жңүдәҶzzе’ҢfиҝҷдёӨдёӘе‘Ҫд»ӨпјҢе°ұиғҪеҝ«йҖҹеңЁдёҖдёӘеҚҒеҲҶеӨҚжқӮзҡ„ж–Үд»¶зі»з»ҹдёӯеҜјиҲӘ
    жӯӨеӨ–пјҢshellиҝҳеҸҜд»ҘеңЁд»»дҪ•дёҖдёӘең°ж–№дҪҝз”Ёcdе‘Ҫд»Өеҝ«йҖҹеҲ°иҫҫеҸҰдёҖдёӘең°ж–№пјҢиҖҢдёҚз®ЎиҝҷдёӨдёӘең°ж–№зӣёйҡ”еӨҡиҝңпјҢrangerд№ҹеҸҜд»ҘзӣҙжҺҘиҫ“е…ҘcdпјҢз„¶еҗҺиҫ“е…Ҙи·Ҝеҫ„пјҢе°ұиғҪеҲ°иҫҫжҢҮе®ҡзҡ„и·Ҝеҫ„пјҢиҝҷдёӘеҠҹиғҪе°ұеғҸgиҝҷдёӘжҢүй”®дёҖж ·пјҢеҸҜд»Ҙиҝ…йҖҹзҰ»ејҖеҪ“еүҚдҪҚзҪ®иҖҢдёҚз»ҸиҝҮе®ғзҡ„дёҠзә§зӣ®еҪ•
